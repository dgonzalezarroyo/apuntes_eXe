eXelearning Project

  • It is a free software tool, available for Linux, Windows and MAC. It allows you to create multimedia educational resources, without being an expert on HTML or XML.

  • Initially created by the New Zealand Government, today it is supported by several educational public institutions  (Ministerio Educación, País Vasco, Andalucía, Galicia, Madrid y Murcia) and some private companies (Open Phoenix and Toughra Technologies).
