Exporting Formats

Exelearning projects are saved using .elp (eXe Learning Project) files. We can only open them with the eXelearning software. If we want to open them using a different application, we will have to export them using several formats:

  • Educational standar
    • Common Cartridge
    • SCORM
    • IMS Content Package
  • Web Site:
    • Self-contained folder
    • Single page
  • EPUB
  • Text file

And also other ones. To do that, go to the top bar and click on the File menu, that allows saving and exporting contents:
